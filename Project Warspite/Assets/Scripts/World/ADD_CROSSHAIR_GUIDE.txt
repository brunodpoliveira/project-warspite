═══════════════════════════════════════════════════════════════
  HOW TO ADD TURNING CROSSHAIR TO ENEMY
═══════════════════════════════════════════════════════════════

METHOD 1: AUTO-SETUP (EASIEST)
-------------------------------
1. Select enemy in Hierarchy
2. Add Component → Turning Crosshair (Warspite.UI)
3. Done! Crosshair auto-creates itself

The crosshair will:
✓ Auto-find the EnemyLogic component
✓ Auto-create the UI canvas and images
✓ Auto-create the reload indicator
✓ Position itself above the enemy
✓ Show firing cadence (spinning = charging, locked = ready)
✓ Show reload progress (orange circle fills up)


METHOD 2: MANUAL SETUP (MORE CONTROL)
--------------------------------------
1. Select enemy in Hierarchy
2. Add Component → Turning Crosshair
3. In Inspector, configure:
   - Enemy: (auto-finds, or drag EnemyLogic)
   - Offset: (0, 2, 0) - height above enemy
   - Ready Color: Green
   - Charging Color: Yellow
   - Firing Color: Red
   - Crosshair Size: 50


WHAT YOU'LL SEE IN PLAY MODE
-----------------------------
✓ Green X above enemy = Ready to fire
✓ Yellow spinning X = Charging (time until next shot)
✓ Red X = Just fired
✓ Orange circle filling = Reloading


TROUBLESHOOTING
---------------
❌ Crosshair doesn't appear
   → Check enemy has EnemyLogic component
   → Check Main Camera exists in scene
   → Enter/exit Play mode

❌ Crosshair is too small/big
   → Select enemy → TurningCrosshair component
   → Adjust "Crosshair Size" (default 50)

❌ Crosshair is in wrong position
   → Select enemy → TurningCrosshair component
   → Adjust "Offset" Y value (higher = more above enemy)

❌ Crosshair doesn't spin
   → Check enemy is firing (has ammo, has target)
   → Check "Spin Speed" is not 0 (default 360)


ADDING TO PREFAB
----------------
1. Open enemy prefab (double-click in Project)
2. Add Component → Turning Crosshair
3. Save prefab (Ctrl+S)
4. All instances now have crosshair!


REMOVING CROSSHAIR
------------------
1. Select enemy
2. Find TurningCrosshair component
3. Right-click → Remove Component
4. Crosshair UI will be destroyed automatically

  CROSSHAIR VISUAL GUIDE
═══════════════════════════════════════════════════════════════

    ╲  ╱
     ╳   ← Green, not spinning
    ╱  ╲

CHARGING (0.5 sec until fire):
    ╲  ╱
     ╳   ← Yellow, spinning clockwise
    ╱  ╲

JUST FIRED:
    ╲  ╱
     ╳   ← Red, spinning fast
    ╱  ╲

     ◐   ← Orange circle, filling clockwise
          (shows reload progress 0-100%)
